## А что если перевести примеры книги "Грокаем алгоритмы" с языка Python на язык 1С

Решил наполнить репозиторий внешними обработками с примерами из книги автора Адитья Бхаргава
"Грокаем алгоритмы"

1. Первая обработка - Бинарный поиск
Обработка содержит форму с двумя полями "Конечное значение в массиве" и "Искомое значение"
Ниже 3 команды:
- Найти значение: Создает простой упорядоченный массив и вызывает функцию бинарного поиска
- Найти значение в массиве не четных чисел: Создает упорядоченный массив не четных чисел и вызывает функцию бинарного поиска
- Найти значение в массиве четных чисел: Создает упорядоченный массив четных чисел и вызывает функцию бинарного поиска

Чтобы повторить в точности пример из книги необходимо ввести:
 Конечное значение в массиве - 9
 Искомое значение - 3
 Нажать на команду - Найти значение в массиве не четных чисел

Исходный код с комментариями

2. Вторая обработка - Сортировка выбором
Обработка содержит форму с 3 командами;
- Сортировка массива как в книге: создает массив и сортирует в точности как описано во втором разделе книги.
- Сортировка выбором альтернативная: вместо того чтобы удалять значения исходного массива, обновляет текущий минимум неотсортированной части массива
- Создать массив: просто демонстрирует исходный массив, которы в точности повторяет пример из книги.

Исходный код с комментариями

3. Обработка под номером три - Рекурсия
Обработка содержит две группы:
- Группа Вычислить факториал: имеет поле ввода для внесения вычисляемого значения и команду расчитать.
  По нажатию срабатывает рекурсивная функция расчета, после выводится сообщение с результатом.
- Группа Обратный отсчет: имеет поле ввода для начального значения отсчета и команду запустить обратный отсчет.
  По нажатию срабатывает рекурсивная функция которая последовательно выводит значения отсчета от заданного в поле ввода до "0".

Исходный код с комментариями

4. Четвертая обработка - Быстрая сортировка
Обработка содержит две команды
- Быстрая сортировка: создает массив и сортирует в рекурсии, выводит значения итогового массива на экран
- Демонстрация исходного массива
